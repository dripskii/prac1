# Рево-кофе

Проект "Рево-кофе" — это веб-сайт, посвящённый продаже различных сортов кофе и кофейных наборов. На сайте представлена информация о происхождении кофе, его качестве, различных сортах и способах приготовления. Пользователи могут выбрать кофе по вкусу и приобрести его через интернет-магазин.

## Используемые технологии

- **HTML5** — для создания структуры веб-страницы.

## Особенности проекта

- Разделение информации на несколько секций, каждая из которых отображает различные аспекты кофейного мира: происхождение, качество, зерна, и способы приготовления.
- Возможность покупки кофе и наборов через кнопки "Купить" и "Подробности".
